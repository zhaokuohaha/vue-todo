# Vue-Todo

线上地址: [todo.oldzeng.com](http://todo.oldzeng.com)

使用`vue` 开发的`Todo List`单页应用, 完全依赖`local storage`进行数据存储, 不会记录用户信息, 没有网络请求.


### 功能:
使用`pwa`模板构建, 支持离线使用
